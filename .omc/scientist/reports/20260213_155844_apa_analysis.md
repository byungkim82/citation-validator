## 2. APA 7th Edition Official Rules

### 2.2 Title Capitalization: Sentence Case

**Rule:** Capitalize only:
1. First word of title
2. First word after colon (subtitle)
3. Proper nouns

[FINDING] Sentence case improves accessibility and readability (primary rationale for APA 7th)
[STAT:capitalization_rule] 3 capitalization triggers only

**Examples:**
- ✅ `The effects of meditation: A meta-analysis`

**Common errors:**
- Capitalizing every major word (Title Case confusion)
- Failing to capitalize after colon
- Improper noun handling (e.g., lowercasing "COVID-19")

```python
def is_sentence_case(title):
    words = title.split()
    # First word should be capitalized
    if not words[0][0].isupper():
        return False
    
    # Most other words (except after colon, proper nouns) should be lowercase
    capitalized_count = sum(1 for w in words[1:] if w[0].isupper())
    
    # Heuristic: if >30% of non-first words capitalized, likely Title Case
    if capitalized_count / (len(words) - 1) > 0.3:
        return False
    
    return True
```
